DevGoods

Site: TBA

Published: TBA

DevGoods is a community driven and maintained project. A few individuals, including myself, pioneered this project start to finish. The purpose of this project is too provide those in the Front-End Development and Back-End Development field, as well as those just looking for related information, a place to find useful links/resources.

Any links on the website are scraped from the DevTips slack channel and posted to the website, so it is always updating with new information. We want to grow together and teach each other.
